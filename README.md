# Portfolio Simulator
Actuarial Portfolio Assesment Tool

## 📊 O que o projeto faz
- Medidas de posição (média, mediana, moda, mínimo, máximo)
- Medidas de dispersão (variância, desvio padrão, coeficiente de variação, assimetria)
- Detecção de outliers pelo critério de Tukey (1,5 × IQR)
- Teste de aderência à distribuição uniforme (Qui-Quadrado)
- Anexo técnico com todos os cálculos passo a passo

## 🛠️ Tecnologias
- Python
- Streamlit
- Pandas
- NumPy
- SciPy
- Matplotlib
